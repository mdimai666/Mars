# Кеширование

## Полное кеширование страницы

Используется как аргумент страницы `@cache "20m"` (Кеширование на 20 минут). 

Поддерживается h - час, m - минута s - секунда. 

### Пример
```hbs
@page "/my-page"
@cache "20m"

<h1>{{Title}}</div>
```

> @cache - работает только в Production среде, при локальной разработке может не работать.

Принудительное кеширование, обычно требуется для локальной разработки
`@cache-force "20m"`

## Кеширование контекста данных.
Можно использовать кеширование запроса.
```
{{#context key="index-context" cache="20m"}}
posts = ef.post.Take(10).ToList()
{{/context}}
```

И между страницами тоже, если key будет совпадать.

> На странице можно разместить множесто отддельных `{{#context}}`
