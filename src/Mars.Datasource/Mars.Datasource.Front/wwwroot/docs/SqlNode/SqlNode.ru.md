# SqlNode

SqlNode — это узел для выполнения SQL-запросов к базе данных и получения структурированных данных.

Узел позволяет:
- писать произвольный SQL-запрос или использовать параметры,
- выполнять запрос к выбранному источнику данных,
- возвращать результат в виде набора строк или объекта,
- использовать результат запроса в последующих узлах.
