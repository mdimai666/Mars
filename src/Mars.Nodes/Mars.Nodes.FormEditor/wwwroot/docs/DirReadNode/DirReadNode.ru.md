# DirReadNode

Блок для чтения файлов из директории.

## Параметры

| Параметр | Тип | Описание |
|----------|-----|----------|
| `DirPath` | `string` | Путь к директории |
| `Pattern` | `string` | Шаблон фильтрации: `*` — все файлы; `.txt,.json` — только указанные форматы |
| `MaxDepth` | `int` | Глубина обхода: `1` — только корневые; `0` — все вложенные; `2` — включая один уровень вложенности |
| `ReturnRelativePath` | `bool` | Возвращать относительные пути (по умолчанию `true`) |
| `UseRootGitIgnore` | `bool` | Использовать найденный `.gitignore` |
