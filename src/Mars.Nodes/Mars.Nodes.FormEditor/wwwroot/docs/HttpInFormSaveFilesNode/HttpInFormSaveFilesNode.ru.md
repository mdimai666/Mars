# HttpInFormSaveFilesNode

Компонент предназначен для сохранения файлов файлов из формы `IFormCollection` (`multipart/form-data`) в хранилище.

Используется в связке с HttpInNode

## 🚀 Возможности
- **Парсинг шаблонов:** Поддержка динамических плейсхолдеров для дат, времени, уникальных идентификаторов и метаданных файлов.
- **Безопасность (Path Traversal Protection):** Автоматическая очистка имен файлов и полей от запрещенных символов операционной системы (`/`, `\`, `:`, `*` и т.д.).

---

## 📅 Доступные плейсхолдеры в шаблоне

### FilePathTemplate

Динамической генерации путей и безопасного сохранения

Вы можете комбинировать следующие токены в параметре `FilePathTemplate`:

| Токен | Описание | Пример результата |
| :--- | :--- | :--- |
| **Данные файла** | | |
| `{file_name}` | Полное имя файла с расширением | `photo.jpg` |
| `{file_name_only}` | Имя файла без расширения | `photo` |
| `{file_ext}` | Расширение файла с точкой (всегда в нижнем регистре) | `.jpg` |
| `{field_name}` | Имя инпута (property) из HTML-формы | `avatar` |
| `{guid}` | Уникальный 32-символьный хэш (без дефисов) | `d3b07384d113ede...` |
| `{unique_suffix}` | Уникальный суффикс | `20260529_206c1bc9` |
| `{unique_file_name}` | Уникальное имя файла | `photo_20260529_206c1bc9.jpg` |
| **Дата (Date)** | | |
| `{yyyy}` | Год (4 цифры) | `2026` |
| `{yy}` | Год (2 цифры) | `26` |
| `{MM}` | Месяц с ведущим нулём | `05` |
| `{M}` | Месяц без ведущего нуля | `5` |
| `{DD}` | День с ведущим нулём | `09` |
| `{D}` | День без ведущего нуля | `9` |
| **Время (Time)** | | |
| `{HH}` | Час в 24-часовом формате с нулём | `14` |
| `{H}` | Час в 24-часовом формате без нуля | `14` |
| `{mm}` | Минуты с ведущим нулём | `05` |
| `{ss}` | Секунды с ведущим нулём | `34` |
